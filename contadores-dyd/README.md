# Landing D&D Contadores

Este proyecto es una página web de presentación para servicios contables de D&D.

## ¿Qué contiene?

- Sección principal de bienvenida
- Servicios contables
- Sección de análisis
- Llamado a la acción (CTA)
- Formulario de contacto

## ¿Cómo ejecutar el proyecto?

1. Abre la terminal en la carpeta `contadores-dyd`
2. Instala dependencias:

```bash
npm install
```

3. Inicia el proyecto:

```bash
npm run dev
```

4. Abre el enlace local que aparece en la terminal.

## ¿Cómo generar versión final?

```bash
npm run build
```

## Imágenes

Las imágenes de la página están en `public/images`.
Si quieres cambiar una imagen, reemplaza el archivo por otro con el mismo nombre.

## Notas

- Diseño enfocado en imagen corporativa.
